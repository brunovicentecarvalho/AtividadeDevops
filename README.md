## About
Este aplicativo consiste em uma atividade realizada para a matéria de DevOps do curso de Engenharia de Software da Unifil.

## Usage
Para utilizar esse aplicativo é necessário que seja instalado a imagem no dispositivo a partir da chamada do nome crud_app-api. Este aplicativo se consiste em uma entrada em um sistema de gerenciamento de empregados de uma empresa a partir de um simples CRUD.
